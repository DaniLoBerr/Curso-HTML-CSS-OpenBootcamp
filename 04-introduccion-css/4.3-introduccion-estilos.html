Introducción a los estilos en CSS

Colores

Colores predeterminados

Se puede modificar el color del texto con los colores predeterminados que nos ofrece CSS.

Colores hexadecimales

Se puede dar un color al texto a través de su valor haxadecimal. Existen páginas como coolors.co que te muestran los valores hexadecimales de diferentes paletas de colores aleatorias.

Colores RGB

Se puede modificar el color del texto con los valores RGB (Red, Green, Blue). Se da un valor de 0 a 255 de cada uno de los colores y el color resultante es la mezcla de los tres.

Colores RGBA

Igual que los colores RGB pero le añade un cuarto valor que afecta a la opacidad. Se da un valor entre 0 (transparente) y 1 (totalmente opaco).

Fondos

Se puede colorear el fondo de etiquetas contenedoras (como por ejemplo la de este div) con el atributo "backgroud-color" en CSS. Hay muchos tipos de atributos "background", con el "background-color" modificamos solo el color. Se pueden utilizar colores de cualquiera de las maneras anteriores. Por ejemplo, voy a utilizar un RGBA aquí.

También se puede agregar una imagen de fondo con el atributo "background-image" y el valor url(""). Se puede introducir una url local o externa.

Se puede utiliza el atributo "background-position-x" para elegir en qué punto del eje horizontal queremos poner el texto. Con el atributo "background-position-y", modificaríamos el eje vertical. Y con el atributo "background-position", modificaríamos los dos a la vez, dando dos valores del tipo top, center, bottom, left, right (el primer valor correspondería al eje x y el segundo al eje y).

Espacios y Tamaños

Tamaño: alto y ancho

El ancho de un contenedor se puede modificar con el atributo "width". La altura de un contenedor se puede modificar con el atributo "height". Se pueden utilizar por separado y se pueden combinar. Si se expresan por separado, el que no está indicado toma su valor de forma dinámica, en función del contenido. Sus valores se pueden expresar de diferentes maneras, lo habitual es en píxeles. Trabajar con píxeles no es muy responsive por lo que más adelante se verán opciones más recomendables.

Espacios: border, padding y margin

  • El border es es el borde del contenedor. Se le puede aplicar diferentes atributos como el "border-style" (el tipo de borde), el "border-color" (el color del borde), el border-width (el ancho del borde), etc.
  • El padding es el espacio interno que hay entre el contenido y el borde del contenedor. Se puede modificar el padding de cada lado por separado
  • El margin es el espacio externo entre el contenedor y los elementos de su alrededor.

    • De cada elemento de los espacios se les puede aplicar valores por separado para cada lado del contenedor (top, bottom, left, right). Incluso al atributo genérico de cada uno se le puede dar uno, dos, tres o cuatro valores y cada uno de ellos afectará a cada lado del contenedor: si hay dos valores, afectarán al eje x el primero, los lados arriba y abajo, y al eje y el segundo, los lados derecho e izquierdo. Si hay más valores, afectarán a los lados en el sentido de las agujas del reloj.

Textos

Tipo de fuente

El tipo de fuente se determina con el atributo "font-family". Css nos ofrece de manera predeterminada diferentes tipos de fuentes. Se puede cambiar el grosor de la fuente con el atributo "font-weight". 400 es el valor estándar. Se puede cambiar el tamaño de la letra con el "font-size". En el font-size se pueden definir valores de diferentes maneras (píxeles, puntos, tamaños predeterminados como larger, smaller, etc).

Si queremos que los cambios en la fuente afecten a todo el documento, se debe seleccionar la etiqueta html en el documento css.

Se pueden importar diferentes tipos de fuentes externos, como por ejemplo de Google fonts. En esta página, puedes seleccionar el tipo de fuente que quieres y después seleccionar diferentes características del mismo (diferentes grosores de la fuente, la cursiva, la negrita, etc). No se recomienda importar todas las características porque puede ralentizar la página, lo mejor es seleccionar unas pocas características que más o menos creamos que vamos a utilizar e importar solo esas.

Se puede importar la fuente de varias maneras:

  • Copiando en la etiqueta head del documento html una o varias etiquetas link con la dirección donde se encuentre la fuente que queremos utilizar. Habría que copiar también en el atributo font-family del documento CSS el nombre del tipo de fuente.
  • Importando una dirección tipo @import dentro de una etiqueta style dentro de la etiqueta head.
  • Importando directamente en el documento CSS un enlace @import con la dirección de la fuente. Esta es la opción más recomendable para no ensuciar el documento html con cosas relacionadas con el estilo.
