Posicionamiento en CSS

Existen diferentes propiedades para posicionar elementos en una página web. Vamos a ver 5 de ellas:

Static

El posicionamiento estático es el posicionamiento por defecto, el que usa el user agent stylesheet. Con este valor de posicionamiento, no se pueden modificar otras propiedades relacionadas con el posicionamiento de elementos.

Posicionamiento estático (static)

Relative

El posicionamiento relativo consiste en la posición en la que colocamos un elemento en relación a su posición estática, es decir, a como estaría posicionado ese elemento por defecto. Con esta propiedad, podemos utilizar otras propiedades para posicionar los elementos, como left para separar el elemento del lado izquierdo de la página, etc. Esta propiedad de ejemplo no tendría nada que ver y sería complementaría a otras propiedades como el margin y demás, el margin se aplicaría a partir de la separación ya indicada en el left.

Posicionamiento relativo (relative)

Absolute

El posicionamiento absoluto nos permite posicionar un elemento en un punto de nuestra página y que se quede ahí fijado en función del elemento que lo contiene, es decir, si el elemento con esta propiedad lo colocamos en el body de una página, el elemento se colocará donde le indiquemos respecto al elemento body, en este caso la página entera. Pero podemos colocarlo dentro de otro elemento contenedor y este se colocará en la posición que le indiquemos relativa a este elemento que lo contiene. Lo habitual cuando queremos hacer esto es que el elemento con posición absoluta se encuentre dentro de un contenedor con posición relativa.

Con este posicionamiento, se elimina el espacio sobrante del contenedor y se representa nada más el espacio que ocupa el contenido.

Posicionamiento absoluto (absolute)

Fixed

Posicionamiento que funciona igual que una posición absoluta en la que además nos fija el elemento en un punto concreto de la pantalla (en relación al viewport), es decir, hagamos lo que hagamos, este elemento aparecerá siempre fijado en un punto de la pantalla. Al igual que en el poscionamiento absoluto, se elimina el espacio sobrante y se representa nada más el espacio que ocupa el contenido.

En este tipo de posicionamiento se suele utilizar la propiedad z-index, que se utiliza para poner ciertos elementos por encima de otros.

Posicionamiento fijado (fixed)

Sticky

Posicionamiento que se basa en el relativo y en el fijo. Con este posicionamiento lo que hacemos es que un elemento tenga una posición relativa dentro de la página y que cuando nos movamos por ella, hagamos scroll o lo que sea, el elemento se mueva con nostros y se quede fijado en el punto que le hayamos indicado. Se suele utilizar para, por ejemplo, barras de navegación que queramos tener siempre presente cuando nos movamos por la página.

Posicionamiento pegajoso (sticky)
